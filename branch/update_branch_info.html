Update Branch Info

Follow the steps below to update branch info: -
At First you need to log into the software successfully.

First select the payroll from the administration.
A new list will then appear.

Step-1
After selecting the section, a new list will appear.

Select the Branch section from the Human Resources section of the list.

A new list will appear after selecting the section.

Step-2
Select the Updated Branch Information section from the list.

clicking on section, a list of all the branches will be displayed.

Step-3
If you want to close the branch, click the close button.

Step-4
Click the Edit button to update the branch's new information.
A new form will appear after clicking the Update button.
Fill out the form correctly.

Step-5
When done, click the Update button.

Step-6
If you want to close the form, click the close button.
